## 🚀 Getting Started

To run this project locally:

```bash
# 1. Clone the repository
$ git clone https://github.com/itsyoboysumit/portfolio-react.git

# 2. Navigate into the directory
$ cd portfolio-react

# 3. Install dependencies
$ npm install

# 4. Start the dev server
$ npm run dev
```
